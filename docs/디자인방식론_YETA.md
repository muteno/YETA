# 디자인 방식론 — YETA (값 아닌 계승·통일·픽토그램 규율)

> **위상**: 디자인 *방식론*(레포 달라도 통하는 원칙) 이식본. **값 축**(토큰 hex·팔레트·컴포넌트 실물)은 `viewer/index.html` `:root`·`docs/CII_컴포넌트계승인덱스.md`가 정본 — 이 문서는 "어떻게 계승·통일하는가"의 원칙만 담고, 각 원칙의 YETA 실물은 링크로 가리킨다.
> **경계 한 줄**: 그 문장에서 특정 hex·토큰명·셀렉터를 지워도 규칙이 성립하면 = 방식론(이 문서) · 지우면 껍데기면 = 값 축(`:root`/CII).
> **충돌 시 최종 정본**: 값 = `viewer/index.html :root` · 규칙 = CLAUDE.md + `docs/절대명령2_정본인덱스.md`. 이 문서와 어긋나면 그쪽이 이긴다.

## 0. 표기
- 넘버링 4단(`1.`→`1)`→`a)`→`가)`) · 강조 = 백틱·별표만 · 본문 날짜 스탬프 금지(경위는 `docs/작업이력.md`).

## 1. 계승이 디폴트
- a) 새 UI/UX는 새로 디자인하지 말고 **기존 토큰·컴포넌트를 계승**한다. 계승이 디폴트라 매번 "계승" 선언은 안 하고, 말하는 건 예외뿐. 일관성이 품질이고 재설계 반복은 드리프트·피로를 부른다. → YETA 진입점: `viewer/index.html` `:root`(값)·`docs/CII_컴포넌트계승인덱스.md`(컴포넌트)·`AGENTS.md` §🎨.
- b) **계승 소스 = 생김새가 아니라 배치 맥락(형제 유무·표면)으로 고른다**: 여러 개 나란한 바를 전제한 텍스트-온리 칩을 형제 없이 단독 배치하면 어포던스 0의 고아 라벨이 된다 → 그 표면의 버튼 패밀리(배경·테두리·눌림 있는)를 계승해 어포던스 확보. 버튼·눌림 패턴 정본 = `구성도/00_가이드북_버튼인터랙션.html`.

## 2. 픽토그램 — 표지판 도형은 SVG · 정렬은 시각중심 실측
- a) **표지판성 도형(상하좌우 화살표·삼각형·X·체크)은 유니코드 문자로 찍지 마라** → SVG path만. 폰트 글리프는 원/박스 정중앙에서 편심돼 정렬이 깨진다(대칭 viewBox path만 정중앙 보장). 문자 렌더 허용 = 순수 텍스트·숫자뿐. 잔존 문자 도형은 발견 즉시 교체. → 아이콘 SSOT = `viewer/nm-svg.js`(인라인 재선언 = 게이트 차단) · 시각 정본 = `구성도/픽토그램.html`.
- b) **아이콘 정렬 기준 = 셀(버튼 박스)이 아니라 그 안 픽토그램(path) 시각중심**. 정렬 지배 값들(컨/헤더/행 padding·셀폭)은 연쇄 계산이라 하나 바꾸면 전부 재검증. 검증은 눈짐작·스크린샷 금지 → path `getBBox` 시각중심을 절대좌표로 환산해 픽셀 실측, 1px 초과일 때만 손댐(서브픽셀 잔여는 홀짝 반올림이라 불가피).

## 3. 색 규율 — 기존 유지 · 무채색 크롬
- a) **기존에 이미 있는 색은 그대로 유지가 원칙**. 새 색 창작이나 기존 색 제거는 임의 금지 → 운영자 확인. 팔레트 정본 = `docs/브랜드_팔레트_확립본.html` · 메뉴별 대표색 = `구성도/메뉴별 대표색.html`.
- b) **단 '통일'은 예외 = 묻지 말고 즉시**: 팔레트 밖으로 튄 아웃라이어(상속 leak·제각각 raw·형제와 어긋난 틴트)를 기존/형제 색에 맞춰 정렬하는 건 계승 디폴트. 확인이 걸리는 건 팔레트에 없는 새 색 창작이나, 의도된 색을 없애 통일을 깨는 경우뿐.
- c) **무채색 크롬**: 중립이어야 하는 팝업·메뉴·드롭다운은 무채색 — 색 틴트·뒤 배경색 비침 금지. 글래스(blur)는 유지하되 충분히 불투명 + 필요시 채도 0으로 뒤 색 차단. 예외 = 색이 '의미'인 요소(상태색·on-state·콘텐츠). 규칙은 크롬 한정.

## 4. 인터랙션 — 반복 인터랙션 = 단일 표준 + 위임 1핸들러
- a) 반복되는 인터랙션(다운로드 버튼·아이콘 누름 모션 등)은 표면마다 새로 만들지 말고 **단일 표준 클래스 + 위임(캡처) 1핸들러**로 처리한다. 새 인스턴스는 표준만 붙이면 색·효과·모션을 자동 상속 — 개별 창작은 드리프트. 값 정본은 한 곳, 위치·크기만 컨텍스트 클래스로 변주. → 눌림 scale = `--press-*` 토큰 사다리 · 패턴 정본 = `구성도/00_가이드북_버튼인터랙션.html`.

## 5. 통일 — 최외곽/형제 표면을 한 값으로
- a) '통일'은 **최외곽/형제 표면을 한 값으로 맞추는 작업**. 각 뷰의 제일 바깥 박스는 단일 값으로 통일(뷰마다 다른 값 창작 금지), 내부 중첩 박스는 각자 값 유지(패딩 인셋이라 코너 충돌 없음). 한 행에 같은 성격 요소가 두 스타일로 혼재 금지. 반복 요소(닫기 등) 제각각 크기도 점진 통일. 별도 레이어(모달 등)는 별도 통일값 허용. → 시각 정본 = `구성도/표면 테두리 강조.html`·`구성도/버튼 배치.html`.

## 6. 머리·타이포 위계
- a) 머리·제목 위계는 **전역 통일된 소수 계층**으로: 대주제 → 부제(무기호 강조 = 굵게/강조색만) → 소머리(불릿 `•` 하나로 전역 통일). 크기·굵기·색은 리터럴이 아니라 **토큰으로 박아** 하나 고치면 전파. → 정본 = `구성도/텍스트 위계.html`.
- b) **헤더·타이틀·부제·버튼 라벨은 텍스트 단독 또는 픽토그램 단독** — 한 줄에 아이콘+글자 공존 금지(상태 변화는 교체). 외부링크 등 어포던스는 장식 pseudo-element SVG로만 덧댐. 어포던스를 통일·제거할 땐 대체를 먼저 마련 — 그냥 없애 기능 저하 금지.

## 7. 기하 — 뷰포트 @media 분기 금지 (앵커가 px 고정이면)
- a) 뷰포트별 `@media` 분기는 앵커 요소가 px 고정일 때 금지 — 종속값(스크림·오프셋 등)도 단일값이어야 정합. 한쪽 뷰포트만 조정하면 다른 쪽이 방치돼 회귀한다. 분기는 앵커 자체가 뷰포트 가변일 때만.

## 8. 승인 경계 — 신규 UI 요소 '화면 등장' = 논블로킹 사후 확인
- a) 라이브 화면에 **새로 생기는 요소**(새 버튼·칩·배지·토글)는 컴포넌트 계승과 **별개 축** — 묻지 말고 진행하되 3중 안전장치: {① 그 요소만 단일 커밋 분리(롤백 = git revert 한 줄) ② 보고에 ⚠배치 딱지 + 어디에·어떤 모양 1줄 ③ "이 위치·모양으로 해석했다 — 의도 맞나?"를 묻기 전에 먼저 붙인다}. **기능을 한 줄로 승인받았어도 그건 기능 승인이지 배치 승인이 아니다.** 배치 후보가 명백히 갈리면(2+안) 플레이그라운드(HTML 갤러리)로 고르게 한다. 스코프 = '새로 등장'하는 요소만 — 기존 요소 계승·수정·통일은 안 물음(= CLAUDE.md [4]·[5]와 동일 축).

## 9. 공통 UI 로직 — 디자인 시스템과 별개로 항상 적용
- a) **어포던스가 설명한다**: 컨트롤 옆 사용법 산문(힌트 줄·안내 캡션)은 기본값 = 넣지 않음 — 라벨·상태색·픽토그램·비활성 dim이 스스로 설명하게. 빈 상태(빈 리스트·패널)는 필러 문자열 금지, 조용한 공백이 디폴트(예외 = 사용자 행위 피드백 1줄: 결과 없음·불러오는 중·에러). 설명 넣고 싶어지면 = 어포던스 부족 신호 → 문구가 아니라 컴포넌트를 고친다.
- b) **번쩍임(flicker) 방지**: ① 입력칸 담은 컨테이너·오버레이엔 `backdrop-filter` 금지(리페인트마다 재샘플 = 모바일 GPU 번쩍) ② 아이콘 갱신은 `innerHTML` 직접 재대입 말고 가드 세터 경유(SVG 노드 재생성이 타이핑마다 번쩍) ③ 비동기 처리중 표시는 전체화면 딤 대신 타이틀 옆 인라인 점모션.
- c) **자유입력칸 = OS 자동완성/자동교정 끔이 디폴트**(`autocomplete`·`autocapitalize`·`autocorrect`·`spellcheck` 4종 off) — 안 끄면 모바일 OS가 비번·카드·주소 바를 붙여 오입력·번잡. 맞춤법은 앱 자체 로직이 처리. → **하드 게이트**: `shared/check_refs.py`가 4종 누락을 커밋 차단.
- d) **모달 = `history.pushState`/`popstate`로 뒤로가기 흡수**: 열 때 push, `popstate`로 닫기 → 폰 뒤로가기 = 직전 화면 복귀(앱 이탈 아님). X/백드롭/ESC로 닫으면 push 항목 정리.
