# 제1명령 (포터블) — 디자인 = 루트 토큰 「계승」 또는 「갱신」

> 어느 레포에나 이식하는 **최상위 디자인 거버넌스**. 15인 교차검증 반영판.
> **(A) 개념규칙 = verbatim 복제**(수정 없이 복사) · **(B) 바인딩 표만** 그 레포 값으로 치환 · **(C) 5단계 셋업**.
> 핵심: "지켜지게" 만드는 건 문구 세기가 아니라 **③ token-lock 게이트**다. 이거 없이 (A)만 박으면 과거처럼 안 지켜진다.

---

## (A) 개념규칙 — 복제 대상 (플레이스홀더만 치환, 나머지 그대로)

레포의 **모든 디자인 값**(색·크기·간격·반지름·blur·모션·눌림·그림자)은 예외 없이 둘 중 하나다:

- **① 계승 (디폴트·안 물음)**: `<SSOT>` 의 토큰을 `<계승문법>` 으로 가져온다. 정확 토큰이 없으면 **가장 가까운 토큰 자동 계승**(여기까진 안 물음).
- **② 갱신 (근접조차 없을 때만)**: 운영자에게 **㉮ 갱신(신규 토큰)할지 ㉯ 무엇으로 계승할지(근사 후보 제시)** 를 물어 **승인 후** 진행.
  - 값 미세조정 = 승인 + `<GATE>`. **구조 토큰 신설 = 기틀 = 교차검증**(가벼운 값 조정과 구분).
- 컴포넌트는 값 아님 → `<컴포넌트정본>` 의 정본 셀렉터/레시피 **계승**(별개 축, :root 아님).

**기계강제 = `<LOCK>` (토큰 승인 원장).** `<SSOT>` 에 새 토큰이 생겼는데 `<LOCK>` 에 없으면 `<GATE>` 가 커밋을 막는다(rc=1). 승인 = `<LOCK>` 등재(PR diff 를 운영자가 리뷰). **문구가 아니라 이 게이트가 강제한다.**

**예외 (raw 허용·물음 없음)**: 기하 계산(−50%·원 50%)·키프레임 안무 %·광학 1px 보정·OS 강제(safe-area 등)·로컬 z-index·**토큰 없는 축**(box-shadow 등). = "매 px·그림자마다 안 물음."

**위상**: 디자인 *내용* 축에서 최우선. 라우팅·기틀 교차검증·기존 절차를 **우회하지 않으며**, 실체 정본은 `<디자인계약>`. ⚠️ **초록 게이트 ≠ 전면 준수**(예외·모델 자율분 존재).

## (B) 레포별 바인딩 표 — 이 슬롯만 치환

| 슬롯 | 뜻 | yeta 예시 | 새 레포에서 |
|---|---|---|---|
| `<SSOT>` | 값 단일 출처 | `viewer/index.html :root` | `:root` / `tokens.json` / Style-Dictionary / Tailwind config … |
| `<계승문법>` | 토큰 참조법 | `var(--token)` | `var()` / `theme(...)` / `$token` / `import { token }` … |
| `<LOCK>` | 토큰 승인 원장 | `design-tokens.lock` | 같은 파일명 권장 |
| `<GATE>` | 커밋 게이트 | `python3 shared/check_refs.py` (pre-commit) | CI / pre-commit 스크립트 |
| `<BUILD>` | 거울·파생 재생성 | `shared/build_design_mirror.py build` | 있으면 기입, 없으면 생략 |
| `<컴포넌트정본>` | 컴포넌트 계승처 | `docs/CII_컴포넌트계승인덱스.md` | Storybook / 컴포넌트 인덱스 … |
| `<디자인계약>` | 실체 정본 절 | `CLAUDE.md §🎨` | 그 레포 디자인 규칙 절 |
| `<훅>` | 세션 주입 | `.claude/hooks/design_digest.py` | 동형 SessionStart 훅 |
| `<승인자>` | operator | 레포 운영자 | 동일 |

> 다른 레포는 `:root` 대신 JSON/TS 에 토큰이 살 수 있다 — 그래서 (A)는 `<SSOT>`/`<계승문법>` 로 **SSOT 형식 불문**하게 썼다. 표만 치환하면 방향이 반대인 레포에도 그대로 붙는다.

---

## (C) 새 레포 이식 셋업 (5단계)

1. **SSOT 확정** — 토큰을 한 곳에(:root / `tokens.json` / Style-Dictionary 등). raw 값 산재 금지.
2. **LOCK 부트스트랩** — 현재 토큰명 전부를 `<LOCK>` 에 등재(1줄 1토큰).
3. **GATE 에 token-lock 추가** — `<SSOT>` 토큰명 집합 vs `<LOCK>`; 락 미등재 신규 토큰이면 `rc=1`(커밋 차단). yeta `shared/check_refs.py`의 `check_token_lock()` 를 그대로 이식(정규식 2줄 + 집합 차).
4. **훅 주입** — SessionStart/UI턴에 (A) 요지를 자동 주입(`<훅>`).
5. **CLAUDE.md 최상단** — (A) 삽입 + (B) 표. 끝.

---

## 왜 이 구조인가 (15인 교차검증 수렴)

- **우선권 무한정 금지** — "다른 모든 규칙에 우선"은 기틀 교차검증·라우팅을 우회하는 라이선스가 된다 → "디자인 *내용* 축 최우선, 충돌 시 실체 정본이 이김"으로 스코프.
- **기계강제가 진짜 지렛대** — 선언만 세게 하면 안 지켜진다(과거 반복). `<LOCK>` 게이트 = "승인의 부재를 기계가 잡음".
- **근접 자동계승 보존** — ①정확→②근접 자동(안 물음)→③갱신(물음). 이게 빠지면 물음이 늘어 마비.
- **예외 카브아웃** — "전부 절대금지·둘 뿐"은 실측상 거짓(기하·box-shadow 등 정당 raw 대량). 예외 없으면 마비.
- **얇게 + 훅 동기** — 본문을 §디자인계약·훅과 3중복제하면 드리프트. 최상단은 얇은 선언+포인터, 실체는 한 곳.
