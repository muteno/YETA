단톡·난입 — 아이데이션 + UI 구성안

초대 기반 · 캐릭터가 거절 가능 · UI 난잡 금지(기존 기틀만 계승) · 비용 = 지금과 동일(턴당 호출 1회) — 시안 전체가 제안 딱지

1결론 요약

  • 방을 새로 만들지 않는다 — `합석` 모델. 스레드는 지금처럼 하나(전 캐릭터 맥락 공유가 이미 배선돼 있음), 바뀌는 건 지금 방에 있는 캐릭터 수(1→최대 2)뿐. 카톡은 1:1에 초대하면 새 방을 만들지만, 무음동은 한 공간(찻집)에 사람이 드나드는 세계라 합석이 정체성에 맞음.
  • 초대 = 이미 예약해 둔 + 버튼(운영자 260705 #13 `향후 기능 진입점`) → 하단 시트에서 캐릭터 탭 → 캐릭터가 카드·시각·관계로 수락/거절을 스스로 판단(거절이 사람다움의 하이라이트).
  • 난입 = 드문 이벤트. 질투 메타(riv)·최근 언급을 트리거로 하루 1회 상한. 유저는 원탭으로 내보낼 수 있음(거절의 대칭).
  • 비용 불변. 단톡이어도 턴당 dispatch 1회 — 한 호출이 최대 2대사 대본을 내고 러너가 화자별 턴으로 분할.

5미세조정 (규칙 다이얼)

방 정원 (나 제외 캐릭터 수)

운영자 말 그대로 `한 명 정도는` = 기본 2명. 3명은 대본 파서·비용 동일하나 수다 밀도가 급증 — 후속 실험 축.

난입 빈도

트리거 = 최근 대화에 그 캐릭터 언급 · 질투 메타(riv) · 데일리 무드. 결정적 시드라 재현 가능, 상한은 세션 마커로 강제.

초대 거절 판정

거절 사유는 살아있는 근거로 — 백은 순찰 시간대엔 안 오고, 윤은 방송 중(0~3시)엔 못 옴. 거절한 캐릭터는 나중에 먼저 난입할 확률↑(떡밥 회수).

턴당 대사 수 (단톡 시)

한 호출이 대본으로 최대 2대사(서로 티격태격) — 러너가 화자별 턴으로 분할. 비용은 1대사와 동일.

내보내기 방식

난입 직후에만 원탭 pill이 잠깐 떴다가 다음 내 메시지 후 자동 소멸(난잡 방지). 언제든 대화로도 가능(`이제 둘이 있고 싶은데` → 캐릭터가 알아서 나감).

7사람다움 포인트 (분신술 아이데이션 연계)

  • 거절이 콘텐츠다. 사유가 캐릭터 카드·시각·관계LV에서 나옴 — 새벽 3시엔 방송 끝난 윤만 오고, 백은 순찰이 끝나야 온다. 거절당한 초대는 그 캐릭터가 나중에 먼저 난입해 회수(`아까는 미안했다`).
  • 질투가 문을 연다. riv 메타 보유 캐릭터가 난입 1순위 — `요즘 걔랑만 놀더라` 같은 등장 대사가 자동으로 성립.
  • 셋이어도 비밀은 둘의 것. 합류자는 최근 흐름(HIST 8턴)만 알고 NOTE:ME(둘만의 기억)는 모름 — 이미 배선된 규칙 그대로라 단톡에서 관계 서사가 안 깨짐.
  • 나가는 것도 스스로. 대화 소강·새벽이 깊으면 캐릭터가 먼저 `나 가볼게` 하고 빠짐(퇴장 sys) — 유저가 방 관리를 안 해도 방이 살아 움직임.
