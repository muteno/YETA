5대안 · 새 아이데이션

대안 A (현 구현) — 픽토 2개 = 고정 + 나가기. 팝업 내용 전체를 트레이로 노출 = 팝업 소멸. 가장 단순·카톡 결.
대안 B — 픽토 2개 = 나가기 + 더보기(⋯). 더보기 누르면 기존 팝업(고정 등 확장 액션)이 뜸. 향후 액션이 3개+로 늘 때 확장 여지.
대안 C — 실시간 추종(손가락 따라 트레이가 열리는 러버밴드). 더 물성감 있으나 구현 복잡·저사양 저크 리스크. 현재는 임계 스냅으로 안정 우선.

기본 채택 = 대안 A. 픽토 매핑/폭/방향을 바꾸고 싶으면 말해줘 — 트레이 폭(60px)·임계(44px)·픽토 구성은 한 줄 조정.
