진입 다이브 플레이그라운드

PIN 승인 → 눈 방향 줌 다이브 + 매트릭스 코드 레인 + VHS zap → 메뉴 진입. 전 구간 dial 세분화 — 매트릭스 고급화가 주인공.

메뉴 진입 — 계단 등장

프리셋

① 다이브 줌·어둠
② 매트릭스 기본
③ 매트릭스 고급
④ 전환 (zap·언락)

선택값 (복사해서 답장에)


    
· 배경 = 영상 정지 프레임 근사(실물은 영상 재생 + 로고 전격 플리커 병행 — 로고는 몸체 1겹 재현).

· PIN 4원 파동·자물쇠 박살 = 고정 맥락 재현(dial 아님).

· 깊이 레이어 = 뒤층이 작고·느리고·어두운 파랄락스(2~3층 = 고급감 핵심).

· 뮤테이션 = 떨어지는 중 글자가 개별 랜덤 교체(0 = 현행 일괄 시프트). 속도 편차·성김·지터 = 균일함 제거(유기적).

· 헤드 광채·글로우(canvas shadowBlur)는 저사양 기기에서 프레임 저하 가능 — 1.2초 한정이라 대체로 무난.

· 0.25×/0.5× 배속으로 디테일 확인 후 1×로 최종 판정. 값 확정하면 실물 배선한다.
