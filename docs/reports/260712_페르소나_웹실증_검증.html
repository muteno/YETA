페르소나 알고리즘 — 웹 실증 검증 + 개선 7

"이 페르소나 알고리즘이 yeta의 핵심 · 진짜같아야 한다"(운영자) — 우리 설계를 학술·업계 실증과 대조했다.

범위 성격모델 · 친밀도시스템 · 상처회복심리 · 페르소나일관성 · 호칭언어 판정 대체로 실증 뒷받침 · 개선 7개 도출

결론. yeta의 [2축 역행·완충·관통·히스테리시스·저작렌즈·티어토큰·상태블록·호칭라더]는 학술·업계 실증과 정확히 정합한다 — 임의 설계가 아니라 선례가 있다. 5인이 공통으로 짚은 개선 7개(A~G)가 이 리서치의 실질 수확이다.

2 개선 7개 (설계 반영)

★ A · 관통 = 의도 게이트

완충을 뚫는 기준은 상처 크기(SHOCK)만이 아니라 "의도(고의성)"다. Fehr 메타분석(175연구·26k명): 용서 억제 최강 요인 = 고의성 r=−.49. → 실수는 완충·빠른 회복, 고의(가치관 위반·배신)만 관통·깊고 느린 회복. 티어토큰에 <<HURT:betray>>(고의) vs <<HURT:slight>>(실수) 구분 = 이미 반영된 축을 의도 기준으로 명문화.

★ B · 역행 = 처벌 아닌 서운함

Persona 3의 관계 역행(리버스/브레이크)은 가장 사실적이었으나 "상호작용이 불쾌"해서 P4 완화·P5 전면 삭제. AI컴패니언 감정조종(죄책감·만류)은 참여 14배↑지만 동력이 분노 → 이탈. → 역행은 캐릭터의 자연스런 서운함이어야지 유저를 벌주거나 죄책감으로 붙잡는 dark pattern이면 안 된다. UX 위원 "온기 계약"의 외부 실증.

★ C · 호칭 역행 2트랙 (애니어그램 직결)

관계 악화 언어는 한 방향이 아니라 두 갈래(demand-withdraw, PNAS): 폭발형(공격: 너→야→욕) vs 냉각형(철회·재격식화: 반말→존댓말 복원→무호칭). 한국어에서 존댓말 복원 = 최고 냉대. 현 라더(너→야→무호칭)는 폭발형만 있었다.

트랙역행 라더유형(배정 연동)
폭발형이름 → 너 → 야 → (욕·독설)8도전·7열정·(3성취)
냉각형이름 → 너 → 존댓말 복원 → 무호칭·침묵5탐구·9평화·1개혁·6충성·4개인

+ 상승 라더도 4단 보강: 씨 → 이름 → 이름+아/야 → 애칭. ⚠️ "너"는 중립 아닌 1단 냉각으로(한국어는 친해도 이름). 전환은 계기 이벤트(트리거 대사) 동반해야 자연스럽다.

D · 완충=잔고 · 사과=부분회복

완충 = 관계 sentiment 잔고(Gottman 5:1 앵커) — 고관계는 중립도 호의로 override, 저관계는 부정 override. 사과는 부분 회복(r≈.42)이지 완전 리셋 아님 → 회복모델 ㉮에 "사과=부분 상쇄" 반영. 단 5:1은 앵커일 뿐 상수 아님(재현성 논란).

E · 상태블록 말미 · L0 Narrator Mode

① 상태블록(yHurt·관계온도·호칭)은 프롬프트 말미(생성 직전)에 둬야 최신성 편향으로 어텐션을 받는다 — 앞부분 카드주입만으론 후반 턴 희석. ② L0 메타차단("나는 AI")을 하드거부→서사 내 우회(Narrator Mode)로 — 하드거부가 오히려 재탈옥 유발(롤플레이 탈옥 89.6%).

F · 관통 등급제 · 영구는 절대선만

Maj'Eyal = 일반 적대는 시간 자동회복 / 아군 살해만 영구. → 우리 관통은 "급락 + 긴 회복로", 영구 단절(절연)은 절대선 위반에만(이미 절연 기각과 정합). 회복로는 항상 존재.

G · 놓친 벡터 (안전·품질)
  • opus 드리프트: 큰 모델일수록 정체성 붕괴↑(arXiv 2412.00804) → 기본 opus 4.8 vs sonnet 다이얼 일관성 A/B 관측 권장.
  • notes 허구 누적: 모델이 지어낸 자기설정이 notes/note_pub 전방공유로 복리 오염 → note 쓰기 검증.
  • 다턴 점진 탈옥: stripMarkers·길이캡은 라운드 누적 공격 못 막음 → 역할-질의 충돌 시 in-fiction 거절(declined 모델을 유해축까지 확장).

4 실증의 한계 (과신 금지)

  • Gottman 5:1 = thin-slicing 재현성 논란 → 앵커로만, 상수 취급 금지.
  • 회복 곡선 = 대부분 횡단 상관(용서 양) — 연속 시간궤적은 측정 아닌 추론.
  • 감쇠 함수(지수·tanh) = 인간 데이터 피팅 아닌 공학 관례("그럴듯함"≠"측정됨").
  • 애니어그램 = 비검증·신비주의 기원 → 저작 스캐폴드로만, "과학적 진실" 취급 금지.
  • "신뢰 10배" 류 = 대중 출처 과장 · 방향성만 신뢰.
