재현·적용 한계
  1. 배경 = 실물 영상 히어로 대신 톤 근사. 색은 브랜드 라임·코발트만(계승 · 무지개색 없음).
  2. 3×3만 번호 유지 매핑(점=키패드 1~9 순서 → 방문 순서=PIN). 4×4는 시각 탐색용 — 16점은 4자리 PIN 매핑이 안 맞아(적용 시 3×3 고정).
  3. 노드 모양·선·이펙트·크기 = 패턴 전용 raw 축(토큰 없는 축). 색·글래스·반지름·모션은 var() 토큰 계승.
  4. 데모 검증(2486)은 로컬 시연 — 실제는 서버 auth 판정. ⚠ 잠금=기틀이라 실제 반영은 확인 후.
