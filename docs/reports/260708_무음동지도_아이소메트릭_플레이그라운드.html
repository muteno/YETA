4재현 한계 각주

· 씬 안 글래스 blur 불가: backdrop-filter × preserve-3d = 크로미움 #323735424 · 파이어폭스 #1952612 열린 버그 — 간판·네임태그는 근흑 알파 가짜 유리로 계승(실장 때도 동일 권장). 진짜 블러는 씬 밖 HUD(선택 카드·복사 바)에만.
· 정사영(무원근) = 아이소메트릭 정석(어느 위치 건물이든 같은 모양). 원근감 원하면 실장 때 perspective 축 추가.
· 3D 변환 위 텍스트는 래스터가 살짝 무를 수 있음 — 간판·태그는 빌보드 역회전으로 축정렬시켜 최소화.
· 주민 = CSS 말(figure). 실장 시 스프라이트(char_*.png)·아바타 칩으로 교체 가능(로드 실패 = 말 폴백 유지).
· 루시(에픽) = places.json 동선표 밖 → 미표시. `에픽 기척` 토글은 위치 주장 없는 연출 제안일 뿐(정본 아님 · 채택 시 동선 편입은 별도 콘텐츠 결정).
· 폰트 = 시스템(뷰어 실장 시 Paperlogy 자동 적용) · reduced-motion = 전 애니 정지 · 저사양 = 김·점멸·펄스부터 감쇠 여지.
· 어깨너머 = 추상 스케일(운영자 확정 — 사람·건물 실물 비율 무시 · 인물 크기 = 줌 연동) · 지도 영역은 touch-action:none(드래그 팬 전용 — 실장 시 지도 탭 전용 화면이라 페이지 스크롤과 무충돌).
· 미니맵 = 실장 시 "지금 대화 중인 방 캐릭터" 마커·알림 점 얹는 확장 축(현재는 위치·뷰파인더만).
