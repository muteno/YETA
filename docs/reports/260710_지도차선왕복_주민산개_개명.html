2. 한국식 왕복 차선 — 운영자 스펙 그대로

운영자 스펙구현
지금 도로를 좌우 각 1차선으로간선 7종 × 정·역방향 = 왕복 14대 (기존 편도 6대)
중앙 180° — 조금이라도 위로 가면 우측통행, 아래면 좌측진행벡터 (dx,dy)의 우측 법선 (−dy,dx)으로 중앙선에서 1.2% 오프셋 — 위로 가는 차 = 화면 오른쪽 차선, 아래로 가는 차 = 화면 왼쪽 차선(수학적으로 동일 규칙) → 마주 오는 차가 서로 반대 차선

역방향 차선의 'h'(가림)는 구간 속성이라 역순 시 자동 시프트(yLaneOf) — node 검증으로 정·역 모두 같은 물리 구간이 가려짐 확인. 's'(광장 정지선)는 정점 속성 무시프트 = 양방향 각자 교차로 앞 빨간불 정지. 오프셋 1.6→1.2% = 실그림 도로 반폭(~2.2%) 안에 차 폭까지 수납. 신호등도 새 교차로 위치(48,34)로 이동.
