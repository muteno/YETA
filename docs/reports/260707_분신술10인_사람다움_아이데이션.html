2. 베스트픽 — 티어별 (수렴도 × 효과/비용)

T0 지금 바로 코드 몇 줄·프롬프트 몇 줄 — 스키마 변경 0

  • ① 감정 관성: extract_mat에 직전 mood 추출 1줄 + 프롬프트 1줄("직전 공기 tense — 급변 말고 이어가라") + 가드레일("진심 사과·충격은 곡선을 꺾어도 됨" — 안 넣으면 계속 삐져 있는 역효과)
  • ② STATE_BLOCK: TUNE 뒤 한 자리에 {시각대·요일·계절·달%·데일리 무드 시드(sha256(캐릭터+날짜) = 같은 날 같은 기분·무저장)} — date 1콜 공유·캐시 접두 무손상·~120토큰
  • ③ 화자 귀속 수정(집단 분신의 버그 발견): HIST에서 타 캐릭터 대사를 "세라: …"로 렌더(현재 전부 "너:"로 뭉개짐) + 활성 로스터 이름 주입(은퇴자 소환 방지) → "아까 세라가 그러던데"가 공짜로 열림
  • ④ 태도의 비대칭(순수 프롬프트): 관심 편향 응답(카드 당김/흘림 화제 2줄) · 얼버무리기/되묻기(날조 금지의 정교화 — 거짓 '태도'만) · 선의의 거짓(겉 "괜찮아" / 지문 속마음 — 카드에 이미 내장) · MOOD 연동 웃음 진폭(warm=ㅋㅋㅋ·tense=뚝)

T1 다음 스텝 세션 필드·파서 확장 — 중비용·승수

  • ⑤ sess.rel 공용 상태 블록: {관계LV·affect(무드 캐리오버)·last_seen·hooks_fired·beats} — 관계역학·감정곡선·주도성·망각이 이 한 키에 올라탐. LV0~5(시트에 캐릭터별 정의 완료·미배선)가 드디어 돎
  • ⑥ 휴면·재진입 + 합류 인계: 공백 감지 서운함/사과 + 재뽑기 시 "방금 무디랑 있었지?"(draw가 직전 화자를 이미 앎) + 질투축 급식(등장 빈도 메타만 — 하은7·세라6이 즉시 살아남)
  • ⑦ 기억의 사람다움: NOTE 감정 가중 태그([★]/[·] — 파서 무변경·포화 시 저가중부터 뭉갬) + 회상 트리거(비·음식이 기억 소환 "아 맞다 너 그때…" · 빈도 가드) + `<<PROMISE>>` 약속 원장("그 면접 어떻게 됐어?" 먼저 챙김 — 유일한 실파서 증분 1개)
  • ⑧ 버블 분할 + 타이핑-팝: `\n\n` = 버블 경계 → 카톡처럼 2~3연발 + 글자수 비례 지연 큐(기존 1초 홀드 확장) + few-shot에 지문 심기(코드 0 부스터)

T2 시즌 단위 기틀·저작 대형

  • ⑨ 5레이어 카드 분해(신 9인 카드 저작 시 규격으로): 통짜 ~15k자 → L1 CORE 500자+L2 사전+L3 요약+L4 훅 = ~3k자 — 80% 절감하며 정체성 선명화. 다른 모든 아이디어의 비용을 낮추는 선행 인프라
  • ⑩ 34축 이관 + 성격축 신설: 애착 불안×회피 2축(4분면 창발 — 다가오면 매달리는가 닫는가)·정직-겸손(HEXACO 공백)·감정 기복→회복 속도 매핑 + 방어기제·역린은 카드 필드(유형 = 카드 레일·코드 0)
  • ⑪ 웜-종료 선톡 + governor: 죽기 전 5분의 웜 러너 재활용(한계비용 0) 선톡 + 적응형 백오프(무시당하면 물러남 = 인격) — 구독 원장(웹푸시) 배선과 세트

3. 하지 말 것 — 과설계 경계 (구조 분신의 반론·전 분신 가드 종합)

  • 34축 전량 런타임 배선 금지 — "지능:높게" 같은 기본축 슬라이더 지시는 카드 산문과 싸워 캐릭터를 납작하게 만듦. 기본 12축은 카드에 굽고, 런타임 튠은 유저가 실제 조절할 행동축(태도·관계)만
  • 감정 상태 결정론 금지 — affect는 소프트 힌트만. "mood=blue니까 슬퍼야 함" 하드 오버라이드 = 유저가 방금 한 농담을 못 받는 로봇
  • 스케줄(cron) 선톡 금지 — 정기 "보고싶어" 푸시 = 스팸·조종감. 주도성 = 컨텍스트 반응(귀환·웜 종료)이지 정기 알림이 아님
  • 기억 과구조화 금지 — NOTE 자유텍스트의 LLM 큐레이션이 자연스러움. 망각 곡선 수식 = 과설계
  • 오타-정정·자기 번복은 옵트인 유보 — 품질 규칙(과장 금지·날조 금지)과 충돌면이 커서 화이트리스트(감정·취향만·사실 불변) 설계 후에만
  • 격리 불변식 — 집단 역학 전부 `note_pub`+등장 사실+세계관 관계만. 타 캐릭터의 NOTE:ME는 어떤 경로로도 참조 금지 · 신규 신호는 전부 서버측 파생값(클라이언트 자유텍스트 금지)
  • 과참조 금지 — 시각 주입은 앰비언트("이 시간까지 안 자?")지 시계 낭독("지금 2시 34분이네") 아님 · 회상·질투·강탈은 전부 빈도 캡

5. 추천 착수 순서 (운영자 픽 대기)

  • 1탄(T0 묶음 — 러너 1커밋 + 프롬프트): 감정 관성 + STATE_BLOCK + 화자 귀속 수정 + 태도의 비대칭 프롬프트 — 스키마 무변경으로 "살아있음" 체감 최대 점프. 선행 1건 = 10_세계관 "여름밤" 완곡화
  • 2탄(신 카드 저작과 동시): 페르소나 시트 확정 → 신 9인 카드를 5레이어 규격+훅 뱅크+지문 블록+방어기제 필드로 저작(어차피 새로 쓰는 카드라 한계비용 최소) — 34축 이관·애착 2축도 이때 동승
  • 3탄(세션 스키마 확장): sess.rel + 휴면·재진입 + PROMISE + 버블 분할 — 관계의 깊이
