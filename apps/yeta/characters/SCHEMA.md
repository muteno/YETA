# 캐릭터 상태·계층 스키마 (DB 테이블 정본 · 운영자 260709 "유동 적용")

> `roster.json` = 캐릭터 테이블(뷰어 표시 SSOT). 이 문서 = 그 테이블의 **컬럼 정의 + 계층 판정 규칙** 정본.
> 원칙: **캐릭터별 상태 조정 = roster.json 값 수정만**(코드 무수정). 판정 술어·계층 순서를 바꿀 때만 코드(`viewer/index.html`).

## 1) 상태 컬럼 (roster.json — 전 캐릭터 명시 · 260709 정규화)

| 컬럼 | 타입 | 기본 | 의미 · 효과 |
|---|---|---|---|
| `id` | string | 필수 | 전 시스템 키(스레드·동선·에셋 경로) |
| `name` | string | 필수 | 표시 이름(미대면 시 `???`로 은폐) |
| `met` | bool | `false` | **시즌 시작 개방 박제**. `true` = 처음부터 만난 사이(하서연·루시). `false` = 대화 스레드·합석 이력 생겨야 만남 |
| `locked` | bool | `false` | **스페셜 잠금**(윈터). 활동으로 안 열림 — 특정 조건 해금 · 정렬 최하단 · 난입 후보 제외(러너) |
| `grade` | `''`·`season`·`idol` | `''` | 스페셜 등급 훈장(상호배타 1슬롯) — 배지·링·감정배경. 미대면이어도 배지 노출(티저) |
| `voice` | string | 없음 | `el:<id>` = 프리미엄 음색(보이스 배지 · 등급과 별개 기능 축) |
| `phone` | string | 없음 | Vapi assistant id = 실전화·보이스톡 배선(원거리 = "전화로만 닿아" 경로) |
| `mode` | object | 없음 | 시간대 변신(`{nm, t[]}` — 백·루시) |
| `tune` | array | 필수 | 성향 기본값(L2 · op tune 오버라이드의 베이스) |
| (표시부) | | | `tagline·greeting·greetings·intro·desc·quote·initial·color·avatar·bg·enter_line·stats·job·short·season` |

## 2) 관계 계층 (판정 = 세션 파생 · 뷰어 `viewer/index.html`)

정렬 grp(캐릭터 탭) = 아래 우선순위 위에서부터. 판정 함수 괄호 표기.

| # | 계층 | 판정 | 표기 | 진입 |
|---|---|---|---|---|
| 0 | 대화 중 | 스레드에 턴 존재(`yActiveIds`) | 실명 · "대화 중" | 가능 |
| 1 | 대화 가능 | 기본값 | 실명·카드 전문 | 가능 |
| 2 | 소원 | 만난 사이 + 1일↑ 방치 + `yBond<1.5` (`yEstranged`) | 실명 · 반실루엣 · "만났지만 대화는 어려움" | 차단 — 회복 = 마주침·난입 재회 |
| 3 | 미대면 | `met≠true` + 스레드·합석 이력 없음 (`yUnmet`) | **`???`(강조색)** · 실루엣 · "아직 못 만남" | 차단 — 해금 = 난입·마주침 합석 |
| 4 | 잠금(스페셜) | `locked:true` (`yLockLabel`) | 미대면과 **동일 표기**(260709 통일 · met:true면 "아직 모르는 인물") + 등급 배지 | 차단 — 특정 조건(활동로 안 열림·난입 제외) |

- 친밀도 `yBond` = 내 발화 수 × 감쇠{1일 유지 → 훅 → 이징 · 7일 = 맥스 반감(하한 0.5)}.
- 미대면·잠금 공통: 이름 `???` = `--accent` · 지도 워커 실루엣·탭 차단 · 픽커 은폐(`yUnmet` 단일 판정 — locked 포함 260709).

## 3) 등급 테이블 (`grade` — 훈장 축)

| 값 | 라벨 | 색 | 비고 |
|---|---|---|---|
| `season` | 시즌 | `--season`(보라) | 시즌 주인공(루시) — 배지·감정배경 전용(CTA 금지) |
| `idol` | IDOL | 뉴트럴 `--fg`+글래스(실버) | 스페셜 게스트(윈터) — locked와 조합 가능 |
| (기능) | 보이스 | `--accent` | `voice` 필드 — 등급 아님(기능 아이콘 `.yprem`) |

## 4) 유동 적용 레시피

| 하고 싶은 것 | 수정 |
|---|---|
| 캐릭터 처음부터 개방 | `met: true` |
| 스페셜 잠금 온·오프 | `locked` 토글 |
| 등급 부여·회수 | `grade` 값 |
| 시즌 리셋(전원 미대면) | `met` 일괄 false + 세션 reset |
| 계층 문구·순서 변경 | 코드(`yLockLabel`·`grp`·행 렌더) — 기틀 절차 |
