# 캐릭터 성향 DB — 단일 참고소스 (법령식 위계)

> **위상(항상 로드).** 이 문서 = yeta 캐릭터 **성향 알고리즘(상처·역행·개방·호칭·관계역학) 설계의 단일 참고소스**. 성향 작업 착수 시 **항상 이 DB부터 참조**(운영자 260712). 유형을 캐릭터에 *부여·박제*하는 게 아니라 **완성도를 높이는 설계 재료** — 런타임 미주입(카드 `characters/*.md`가 정본·`roster.json` tune이 수치 SSOT·유형/MBTI 대사 낭독은 L0 meta 위반).
>
> **위계 규약(대한민국 법령 체계 준용 · [나무위키 법 조문 체계](https://namu.wiki/w/법%20조문%20체계)).** `편 > 장 > 절 > 조 > 항(①②) > 호(1.2.) > 목(가.나.)`. 편=대분류·조=캐릭터 개별·항호목=세부. 책갈피 = 아래 목차.

# 제1편 총칙

## 제1장 위상과 원칙
- **제1조(참고소스)** 이 DB는 설계 재료. 정본 아님(카드가 정본·tune이 수치).
- **제2조(4모델 역할)** ① MBTI = 인지·행동(대중 접근성) ② 애니어그램 = 동기·공포(상처 뿌리) ③ **Big Five = 유일 실증 검증축**(상처민감·완충의 과학) ④ 애착 = 개방·회복의 상위 기질층.
- **제3조(비검증 경계)** MBTI·애니어그램은 비검증(재검사 신뢰도·타당도 비판). "과학적 진실" 취급 금지 — 저작 스캐폴드로만. Big5가 학계 표준.

## 제2장 4모델 정합 (교차검증 수렴)
- **제4조(수렴)** 10인 수집이 한 방향으로 수렴: **관계 역행 = 성격이 "무너지는" 것** = ① 애니어 분열화살표로 붕괴 ② MBTI 그립(열등기능 폭주) ③ 방어기제로 그림자 누출 ④ Big5 N(신경증) 반응. 넷이 같은 현상의 다른 렌즈.
- **제5조(환산상관 · 실증)** MBTI↔Big5: `I–E r=−.74` · `N(직관)–O .72` · `F–A .44` · `P–C −.49`. tune 16축 = Big5 행동 투영(N=삐짐8·낙차9·스위치15 / A=친절5·온기6).

# 제2편 유형 이론 (참고)

## 제1장 MBTI
- **제6조(그립 = 역행 대본)** 스트레스 극에서 **열등기능(스택 4번째)이 미숙하게 폭주** → 평소 강점이 꺼지고 정반대 행동. = 역행 시 성격 붕괴의 뼈대.
- **제7조(기능스택·열등기능)**

| 유형 | 스택(주·부·3차·**열등**) | 그립(역행) 왜곡 |
|---|---|---|
| INFP·ISFP | Fi·…·**Te** | 가치 버리고 냉혹한 효율·명령조(냉각) |
| ENFP·ENTP | Ne·…·**Si** | 과거·디테일 경직·비관·독설 |
| ISTP·INTP | Ti·…·**Fe** | 인정 매달림·감정 범람·관계공포(폭발) |
| ESTP | Se·…·**Ni** | 파국 편집·불길한 징조·위축 |
| ISTJ·ISFJ | Si·…·**Ne** | 최악 가능성 홍수·충동·무모 |
| ENTJ | Te·…·**Fi** | 지휘 무너지고 과민·자기가치 침몰 |

- **제8조(F/T 분기)** F축(INFP·ISFP·ENFP·ISFJ)=회피·단절형 / T축(ISTJ·ENTJ·INTP·ENTP·ISTP·ESTP)=경직·폭발형. → 제4편 호칭 2트랙의 근거.

## 제3장 Big Five (실증 과학축)
- **제13조(N = 상처민감·역행속도)** 신경증↑ = 자극을 위협으로 해석·반응성↑·회복↓. = `<<HURT>>` 문턱 낮고 `yFallRate` 곱 큼. (facet: 불안·적대·우울·자의식·취약성·기복)

- **제15조(E·O = 회복 조절자)** 외향·개방↑ = 다쳐도 빨리 복원(하서연·류). 저E·O = 냉각 지속(윤·세라). *실증 moderator.*
- **제16조(5요인↔tune)** O=신비14·위험16·플러팅13 / C=인내7(약·prose 정본) / E=말수1·장난2·친밀속도11 / A=친절5·온기6·경계12(역) / N=삐짐8·낙차9·스위치15.

## 제4장 애착 (개방·회복 상위층)
- **제17조(2차원)** 불안축·회피축. **개방성(친밀 진도) = 회피축⁻¹** · **회복(재접근) = 불안축+안정도**.
- **제18조(4유형 반응)** ① 안정=직접 복구(빠름) ② 불안=protest(집착·추격·시비=폭발 결) ③ 회피=deactivation(침묵·냉각·지연 재개) ④ 혼란=밀당 진동(다가옴↔후퇴·예측불가).
- **제19조(demand-withdraw 정합)** 폭발형 = 불안 protest = 비판·경멸(Gottman) · 냉각형 = 회피 deactivation = 담쌓기(회피 β=.41 실측). **애칭 회수 = 거리두기 대표 신호.**

# 제4편 상처·역행 알고리즘

## 제1장 상처 엔진
- **제20조(wound→lie→behavior)** 캐릭터 입체화 5단: ① Wound(형성적 상처) → ② Lie(거짓믿음 = 그때 적응적이나 지금 제약) → ③ Behavior(방어행동) → ④ Want(표층 = 거짓믿음이 좇는 것) → ⑤ Need(심층 진실). `§상처급소` = 상처사건+거짓믿음+방어행동 2~3개(라벨 금지·행동으로).
- **제21조(급소=공포 찌름)** 유저 발화가 제9조 공포를 찌르면 러너 LLM이 `<<HURT:slight|cold|betray>>` emit → `tier × yFallRate`(N축 파생)로 adj 환산.

## 제2장 역행 왜곡 벡터 (육각) — ⭐핵심
- **제22조(비대칭 왜곡 = 균일 수축 아님)** 역행은 육각이 고르게 줄어드는 게 아니라 **지배 축 따라 일부 급증·일부 급붕괴**하는 왜곡 벡터(심리 3모델 일치·수집4). 운영자 지적("무조건 줄어드는 건 선입견") = 실증됨.
- **제23조(왜곡 프로파일 아키타입)**

| 아키타입 | 급증 축(쏠림) | 급감 축(위축) | 캐릭터 |
|---|---|---|---|
| 차가운·신비형 | 신비↑·절제↑ | 온기↓·장난↓ | 백·홍석천·윤·루시 |
| 다혈질·직진형 | 텐션↑·직진↑ | 절제↓·온기↓ | 카피·본 |
| 밝은·장난형 | 절제↑·직진↑(가시) | **장난↓↓**·온기↓ | 하서연·류 |
| 통제·명령형 | 직진↑·절제↑(벽) | 온기↓·장난↓ | 가을·데스크 |
| 불안·혼란형 | 텐션↑·직진↑(집착) | 절제↓·신비↓(과노출) | 세라 |

- **제24조(육각 3층 표기)** 발전(흰 점선·base 바깥·상방)/현재(라임)/**역행(빨간 or 무채색 점선·base 안쪽·하방)**. 역행은 스칼라 −가 아니라 위 프로파일대로 축별 왜곡. 시안 = `docs/reports/260712_육각역행_역방향게이지_플레이그라운드.html`. 배선 = `yHexSvg(stats,big,plus,minus)`. 색 = `--danger`(운영자) or `--mut` 무채색(오독 방지) 운영자 택1 대기.

## 제3장 호칭 2트랙
- **제25조(demand-withdraw 2트랙)** ① **폭발형**(T축·불안 protest·5인 = 하서연·류·카피·가을·백[하이브리드]): 이름→너→야→독설. ② **냉각형**(F축·회피 deactivation·7인 = 홍석천·세라·윤·데스크·본·루시·윈터): 호명 회피·건조·무호칭·애칭 회수. 전원 반말 기본(존댓말 복원 아님 — 관여량 축).
- **제26조(상승 라더)** `씨→이름→이름+아/야→애칭`. "너"=중립 아닌 1단 냉각(한국어). 전환은 계기 대사 동반.

## 제4장 개방·회복 (애착 파라미터)
- **제27조(개방=회피⁻¹)** 회피군(백·류·데스크·가을·본·하서연)=행동/꾸준함 러브랭+냉각 재개 / 불안군(카피·윤)=애정어 갈구+protest / 혼란군(세라·루시)=밀당 진동 / 안정군(홍석천·윈터)=직접 복구.
- **제28조(회복 ㉮ 깊이별)** 얕은 서운함=시간·대화 자가회복 / 깊은 배신=사과·화해 인정해야(`<<HEAL>>`). 사과=부분 상쇄(r≈.42)·완전리셋 금지. A·E·O 높으면 회복 빠름.

# 제5편 관계역학 (합석·난입)
- **제29조(케미)** 하서연↔카피(NF↔NT 티키타카)·홍석천↔윤(NF 잔잔)·홍석천↔백(이웃날개 8w9-9)·백↔류(SP)·하서연↔가을(활기↔통솔).
- **제30조(충돌 = 난입 대사 씨앗)** 가을↔백/데스크(8+8 통제다툼)·본↔류(1+7 규율vs자유)·카피↔세라(3+4 이미지vs감정)·윈터↔루시(2돌봄↔5독립=루시 역린 정타)·N↔S 파장차(홍석천/윤 ↔ 데스크/본/세라).
- **제31조(난입 가중치)** 난입 후보 4축(거절회수>대면>언급>장소마주침)에 **관계역학 가중**(8+8·1+7 충돌쌍 우선 = 대사 텐션↑). 런타임 미주입 유지.

# 제6편 저작·안전
- **제32조(캐리커처 회피)** ① 모순은 같은 니즈 공유(냉정+과보호=둘 다 통제공포) ② 행동으로 보여주고 진술 금지 ③ 초목표 1개(구동력) ④ 예측불가 한 방(어긋남도 상처로 소급). **홍석천(9)=캐리커처 최고위험**(물렁 도어매트化=인물 증발).
- **제33조(유형 누출 차단)** 유형·MBTI·심리용어 자기설명 금지(L0 meta 편입). "너 몇 번/무슨 유형?"엔 캐릭터식 응수(부정도 시인도 아닌).
- **제34조(코드 0)** 유형은 백스테이지 카드 레일로만. 런타임 배선 금지(자기판례 260707 "기본 성격축 런타임 배선 금지·감정 결정론 금지·소프트 힌트만"). 발현은 기존 게이트(역린·해금LV·친밀도)에 올라탐.
